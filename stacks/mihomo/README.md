# Mihomo

Прокси клиент.

## Референс

- https://wiki.metacubex.one/ru/config/general/
- https://github.com/MetaCubeX/mihomo/tree/Meta

## Развертывание

### 1. Конфигурация

Заполните переменные в файле `vars.yaml` (в качестве примера можно использовать `vars.example.yaml`).

### 2. Запуск

```bash
cd ansible
sudo ansible-playbook playbook.yaml --tags mihomo
```
